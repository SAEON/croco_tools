Anonymous CVS

There is an anonymous CVS server from which the latest versions of
Roms_Agrif can be retrieved. To checkout a copy, do:

cvs -d:pserver:anoncvs@cvs.mpl.ird.fr:/projet login
Use the password "anoncvs"
cvs -d:pserver:anoncvs@cvs.mpl.ird.fr:/projet co Roms_Agrif
