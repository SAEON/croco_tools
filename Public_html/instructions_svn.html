How to get the development version of ROMSAGRIF on Gforge Inria

1- Register on the Gforge Inria

The first is to register on the Gforge INRIA: https://gforge.inria.fr/projects/romsagrif/. We go to the section New account, provide information, and use MOISE (Laurent Debreu is member of this research Team, in LJK, Grenoble) as Inria Team who you are working with.

2- Join the project ROMSAGRIF

The second step is to join the romsagrif project team. On your personal page, search for the project romsagrif, and ask to join it. An email will be send to the administrators of the project.

3- How to download and then update the sources files

  • To download the sources code of this project (ROMSAGRIF source code and Roms_tools matlab files), you have to use (and eventuallty to install it on your computer) Subversion,it is a version control system, also called svn.
    More details can be found on the Inria Gforge help pages, online tutorial and FAQ

    • First you have to do a svn checkout to download the files:

    • The standard users can only execute this via an https procedure:
      svn checkout --username your_login https://scm.gforge.inria.fr/svn/romsagrif
      • Take care of the subversion configuration file : ~/.subversion/servers, in the section concerning the http-proxy :http-proxy-host and http-proxy-port

    • The developpers (users who can upload thier own modifications) can also use the svn+ssh procedure :
      svn checkout svn+ssh://your_login@scm.gforge.inria.fr/svn/romsagrif
      To use this procesdure, you need to configure your ssh-keys (Instructions here)

    In a second time, you have to check about differents updates of the ROMSAGRIF source code and ROMSTOOLS:

    • Simply go to your romsagrif download directory and execute svn update romagrif

  • On Linux, you can use kdesvn for example to manage these updates with subversion: kdesvn but other software exist like TortoiseSVN, RapidSVN, ...

  • These informations more detailed can be found here: http://siteadmin.gforge.inria.fr/FAQ.html

  • The ROMS_AGRIF V2.0-beta developments can be found here:
